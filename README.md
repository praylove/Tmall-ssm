# 项目描述
本项目分为两个模块：前台购物模块和后台管理模块。目前已完成后台管理模块。  
后台管理模块的页面使用了 [ace-admin](http://ace.jeka.by/index.html) 模板, 后台使用 SSM(Spring MVC + Spring + Mybatis) 框架
